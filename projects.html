#2early2read Journal Reading Group


Semi-supervised learning
Semi-supervised learning combines a (small) labelled with a (bigger) unlabelled dataset to perform a task such as classification or segmentation.

Review paper: An Overview of Deep Semi-Supervised Learning (Ouali et al., 2020)

Multi-task learning
Multi-task learning trains a combined network with a separate output head for each task. It is distinguished between hard and soft parameter sharing.

Application paper: A multi-task deep learning model for the classification of Age-related Macular Degeneration (Chen et al., 2018)

Application paper: Multi-task pre-training of deep neural networks for digital pathology (Romain Mormont et al., 2020)

Partial labels
The partial label problem appears when an image is only partly annotated. This affects the performance of a neural network negatively.

Application paper: A Segmentation-Assisted Model for Universal Lesion Detection with Partial Labels (Lyu et al., 2021)

Electronics


Chimera
Chimara is the hybrid model rocket engine, build by Loughborough Space (Loughborough University). A prototype engine was successfully tested. The electronics part focused on opening and closing of a solenoid valve and a valve regulated with a stepper motor. Forthermore, communications with LoRa, a GNSS (GPS, Galileo) module and a 10 DOF sensor were tested. The fuel used for the ground test was HDPE, with gaseous oxygen as ignition medium.

Youtube video: Hybrid Rocket Engine // SUPERSONIC FLOW // Testing 2019 // LboroSpace

Hovering Steward
The goal of the A-levels group project was the development of an autonomous flying waiter. A prototype and concept study for a hex-rotor based logistics system was created including a digital menu for our gastronomy use case. To put it in a nutshell, a drone should be able to serve cupcakes in a restaurant after a waiter selected a table and put the ordered dish in a 3D printed construction on top of the drone.
